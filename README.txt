#IMPORTANT NOTE: 
app is working fine, so if you download setup file
	and install it in C drive (for example Program file x86)
	make sure you run the app as administrator or else 
	ADD/UPDATE/MODIFY button won't work
If you don't want to run as admin, then install it in other drives.


Folder : Python Code : contains raw files
	 that is :
		.py for python script, 
		.ico for icon
		.db for database sqlite3
		.stylesheet for theme
		.ui for ui
Folder : Inventory Manager : contains fully developed application
		just click .exe to open app

Folder : Setup : contains a folder to setup a complete application
	setup file can be distributed easily and then installed
